1. Authenticate to AWS, have a valid `~/.aws/credentials` file setup.
2. Upload an empty file to s3 with the following command:
```
touch filename
aws s3 cp filename s3://some-bucket-name/some/bucket/path/filename --content-type application/json --content-encoding gzip
```
3. Run the following maven commands for this project, update `-Dexec.args` with your S3 bucket name, S3 bucket path/file, S3 bucket region:
```
mvn clean
mvn compile exec:java -Dexec.mainClass="com.jeffbaranski.bugreport.Main" -Dexec.args="some-bucket-name some/bucket/path/filename us-east-1"
```
